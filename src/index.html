Embedded HA Key-Value Store

Fast in-memory etcd-style embedded database with key-value semantics and RAFT consensus for high-availability IoT and edge devices.

Love etcd but hate operating etcd for your appliances and applications? Etclite provides a C API for etcd style key-value stores, with clustering for high availability and automatic persistence to long-term disk storage for recovery purposes.

Etclite is built on dqlite and C-RAFT, an optimised RAFT implementation in C wrapped around sqlite to gain high-performance transactional consensus while preserving sqliteâ€™s famous efficiency and footprint. It adds a layer of key-value capability to emulate enough of the etcd concepts for most embedded applications.
